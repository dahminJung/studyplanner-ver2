# Lotto Number Generator - Project Blueprint

## 1. Overview
사용자에게 무작위로 로또 번호(1~45) 6개를 추천해주는 세련된 웹 사이트입니다. 깔끔한 UI와 부드러운 애니메이션을 통해 즐거운 사용자 경험을 제공합니다.

## 2. Features & Design
- **Random Generation**: 중복 없는 6개의 로또 번호 생성.
- **Modern UI**: 
    - 입체감 있는 번호 공(Ball) 디자인.
    - 부드러운 애니메이션 효과 (숫자가 하나씩 등장).
    - 모바일 대응 반응형 레이아웃.
    - 눈이 편안한 배경색과 세련된 타이포그래피.
- **Accessibility**: 명확한 버튼 색상 대비 및 스크린 리더 호환성 고려.

## 3. Implementation Plan
1. `index.html`: 전체적인 레이아웃 및 웹 컴포넌트 구조 설정.
2. `style.css`: 현대적인 디자인 시스템(컬러셋, 폰트, 애니메이션) 적용.
3. `main.js`: 로또 번호 생성 알고리즘 및 DOM 조작 로직 구현.
